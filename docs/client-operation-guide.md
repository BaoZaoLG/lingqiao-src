# 灵桥客户端操作指南

> 本指南面向普通客户端用户，仅说明日常使用流程。请勿在公开渠道分享卡密、账号、设备信息、错误截图中的个人信息，或任何由管理员单独提供的下载链接和凭据。

## 1. 使用前准备

1. 使用管理员提供的正式安装包或压缩包。
2. 确认电脑可以正常联网。
3. 关闭重复打开的旧版本客户端，避免多个窗口同时运行。
4. 准备好自己的有效卡密。

如安全软件提示拦截，请先确认文件来源。无法确认时，不要继续运行。

## 2. 安装与启动

### 安装包方式

1. 双击安装包。
2. 按安装向导完成安装。
3. 安装完成后，从桌面快捷方式或开始菜单启动客户端。

### 压缩包方式

1. 解压到一个固定目录，例如桌面或常用软件目录。
2. 打开解压后的文件夹。
3. 双击客户端主程序启动。

不要直接在压缩包内运行客户端，否则可能导致配置、更新或运行状态无法正常保存。

## 3. 激活卡密

1. 启动客户端后，在卡密输入框中输入自己的卡密。
2. 点击“激活”。
3. 等待客户端连接服务器验证。
4. 显示“激活成功”后，即可继续使用后续功能。

如果提示卡密无效、已过期或会话失效，请检查是否输入错误；仍无法使用时，联系管理员处理。

## 4. 启动主要功能

1. 确保客户端已激活。
2. 根据界面提示打开目标程序或保持目标程序处于可用状态。
3. 点击客户端中的启动按钮。
4. 等待状态提示变为成功。

使用过程中请不要重复快速点击启动按钮。若客户端提示已启动过或可能存在冲突，请先关闭目标程序和客户端，再重新打开后操作。

## 5. 答题策略

进入题目页面后，先确认题干、选项和输入框已经加载完成，再使用快捷键答题。

### 快捷键

- **启用/停用答题：按 Tab。** 默认处于启用状态。需要临时暂停客户端答题功能时按 Tab；再次按 Tab 可恢复。
- **单题答题：按 F2。** 适合逐题处理。客户端会识别当前题目，获取答案后自动填写到当前题目中。
- **光标反馈：按 F3。** 控制获取答案时是否显示等待光标，不影响答题功能。
- **模型切换：按 F4。** 答案质量不稳定时可切换模型后重试。
- **中键答题：鼠标中键点击题目页面。** 第一次获取答案，答案准备好后再次中键填写。
- **复制答题：选中题干后按 Ctrl+C。** 客户端获取答案后，用鼠标中键点击题目页面填写。
- **取消等待：按 Esc。** 取消中键或复制题目后的等待状态。

### 题型处理

- 单选、多选、判断题会自动选择对应选项；共用选项题、选做题通常按多选/选择类处理。
- 填空、选词填空、完型填空会按空格顺序填写；识别异常时可改用复制答题。
- 简答、名词解释、论述、阅读理解、计算、分录、程序、听力、口语、测评、写作等题型会按文本答案处理，提交前建议人工核对。
- 连线题、排序题或特殊交互题如果页面不是普通选项/输入框，客户端可能只能给出答案内容，需人工完成拖拽、排序或连线。

### 使用建议与异常处理

1. 首次使用某个题库时，先用 F2 测试 1 到 2 道题。
2. 答题时不要频繁刷新、切换页面或快速跳题。
3. 图片、表格、公式或特殊排版题，建议提交前人工核对。
4. 按键无反应时，先点击题目页面空白处；仍无反应时按 Tab 确认答题功能已启用。
5. 长时间等待、填入失败或位置不正确时，按 Esc 取消后重新操作。

## 6. 更新客户端

客户端会在启动或激活后检查新版本。

- 如果出现更新提示，建议点击“立即更新”。
- 如果提示必须更新，请先完成更新后再继续使用。
- 更新下载过程中不要关闭客户端或断开网络。
- 更新完成后，按提示重启客户端或重新打开程序。

如果更新失败，请检查网络后重试；多次失败时联系管理员获取最新安装包。

## 7. 公告与社区

客户端可能会显示管理员发布的版本通知、维护提醒或使用说明，请优先按公告内容操作。

### 使用社区/公共聊天

1. 激活后进入社区或公共聊天区域。
2. 输入要发送的内容。
3. 点击发送。
4. 如发送失败，可根据提示稍后重试。

请勿在社区中发布卡密、联系方式、付款信息、设备信息或包含隐私的截图。若被禁言或限制发言，请联系管理员确认原因。

## 8. 托盘与退出

客户端可能会最小化到系统托盘。

- 如果窗口消失，可在任务栏右下角托盘区域找到客户端图标并重新打开。
- 需要完全退出时，请通过客户端窗口或托盘菜单退出。
- 更换卡密或排查问题前，建议先完全退出客户端再重新启动。

## 9. 常见问题

### 提示“请先激活卡密”

重新输入卡密并点击“激活”。

### 提示“卡密已过期”

联系管理员续期或更换新卡密。

### 提示“会话已失效”

重新激活；如果仍失败，联系管理员。

### 启动功能失败

请按顺序检查：

1. 客户端是否已激活。
2. 网络是否正常。
3. 目标程序是否已经打开或处于异常状态。
4. 是否同时运行了多个客户端窗口。
5. 是否有安全软件拦截。

仍无法解决时，请记录错误提示，并提供给管理员。

### 更新失败

检查网络后重试；仍失败时联系管理员获取最新版本。

## 10. 反馈问题时需要提供的信息

联系管理员时，建议提供以下信息：

- 当前客户端版本。
- 问题发生时间。
- 操作到哪一步出现问题。
- 客户端显示的错误提示。
- 是否刚刚更新、换设备或更换网络。

请不要直接发送完整卡密、账号密码、设备唯一标识或包含隐私信息的截图。如管理员需要核对身份，请按管理员指定方式提供必要信息。

## 11. 安全注意事项

- 只使用管理员提供的正式客户端。
- 不要把卡密借给他人使用。
- 不要公开发布卡密、错误日志或包含个人信息的截图。
- 不要修改客户端文件或使用来源不明的辅助工具。
- 遇到异常收费、冒充管理员或索要敏感信息的情况，请先暂停操作并核实。
